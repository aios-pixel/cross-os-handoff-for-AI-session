# Publication provenance

This repository is a clean public publication of `cn-handoff`.

## What was preserved

- The six canonical source files from each verified snapshot.
- The actual version sequence from `v2.0.0` through `v2.1.7`.
- The functional changes represented by each version.
- Apache-2.0 licensing for the public project.

## Publication-only changes

The following manifest metadata was changed in every public snapshot:

- public author name and profile;
- public repository URL;
- public display and developer names.

These changes do not alter collector or handoff behavior. Public release checksums therefore apply to this repository and are not claimed to be byte-identical to any private manifest.

## What was excluded

- Private Git history and commit identifiers.
- Private project status, handoff, architecture, security, and operations documents.
- Hostnames, usernames, device names, local absolute paths, credentials, tokens, and account data.
- Plugin caches, compiled Python files, test output, and temporary reports.

The public commits and tags are a migration of verified snapshots, not a claim that the original development occurred in this public repository.
